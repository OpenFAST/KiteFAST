KiteMBDyn Preprocessor
======================
This portion of the documentation includes a
:ref:`kitembdynpreprocessor_user_guide` and
:ref:`kitembdynpreprocessor_dev_guide`. The preprocessor theory
documentation is available for download
:download:`here <MBDyn_Preprocessor_Theory.pdf>`. It describes the
mechanisms by which the simplified model described in the YAML file is
converted to a full MBDyn model.

Contact `Rafael Mudafort <mailto:rafael.mudafort@nrel.gov>`_
and `Rick Damiani <mailto:rrdengineering@gmail.com>`_.
